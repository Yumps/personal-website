KBD-Customs

A React Project

Users can build keyboards where they choose options from dropdowns, once a user is happy with their options they can save the build to their "Builds", from there they can view the details of the build with a rough estimate of how much it will cost as well as all the websites to where you can buy the parts from. If the user no longer wants their build they can delete it. A user can also go to the "Shared Builds" page and checkout other users builds and the details.
